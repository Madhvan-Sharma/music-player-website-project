About

This is a Music Streaming Website made by Madhvan Sharma and Himesh Mahabi. This Website uses HTML, CSS, Javascript. This Site also makes use of some frameworks like JQuery, etc. To know more about the Website click the button below.

  • Know More..

  • This Site can play music and their Videos in any known Browsers i.e. Website has been configured to play music in all possible formats like .mp3, .wav, .aac, .ogg, etc. We have also configured the website to play video in any browser in the same way.

    We are able to achieve this compatibility using different source formats. Actually, We have provide audio of the same Music track with different formats. As different Browsers (and Older versions of them) play the audio in different formats. Thus Browser actually checks which audio format it is able to play from top to bottom and the first playable format is Played.
